Linux命令记录

  • man

    • 这条命令必须不顾字母顺序排在最前面。Linux提供了丰富的帮助手册，当你需要查看某个命令的参数时不必到处上网查找，只要man一下即可。 例如，man cat，当然，要进一步了解man的话，直接man man就可以啦。

  • cat

    • 简略的来说，cat主要有三大功能：

    关于cat
    功能 一次显示整个文件 从键盘创建一个文件 将几个文件合并为一个文件
    命令 cat filename cat > filename cat file1 file2 > file

    其中，从键盘创建文件时，只能创建新文件,不能编辑已有文件。当然，如果要详细了解cat的使用规则，是可以man cat的哈。
