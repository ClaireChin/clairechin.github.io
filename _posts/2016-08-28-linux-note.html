Linux命令记录

  • man

    • 这条命令必须不顾字母顺序排在最前面。Linux提供了丰富的帮助手册，当你需要查看某个命令的参数时不必到处上网查找，只要man一下即可。 例如，man cat，当然，要进一步了解man的话，直接man man就可以啦。

  • cat

    • 简略的来说，cat主要有三大功能：

    关于cat
    功能 一次显示整个文件 从键盘创建一个文件 将几个文件合并为一个文件
    命令 cat filename cat > filename cat file1 file2 > file

    其中，从键盘创建文件时，只能创建新文件,不能编辑已有文件。当然，如果要详细了解cat的使用规则，是可以man cat的哈。

  • chmod
  • export

    • 设置或显示环境变量。以我最近的经历为例，我需要使用centos下安装的一个软件what（当然使用某个命令也是一样的道理）， 但这个软件/命令的执行文件不在当前目录，于是我每天创建并登陆虚拟机后都必须指定执行文件的目录，很麻烦， 如果在.bashrc中加入export LD_LIBRARY_PATH="xxx/yyy/..."，就相当于告诉程序，我需要使用what软件时， 需要的文件或相关的一些东西在这个目录里下。

  • echo

    • echo命令的功能是在显示器上显示一段文字，一般起到一个提示的作用。比如，上一条export命令的示例中， 可以使用$echo $LD_LIBRARY_PATH来查看LD_LIBRARY_PATH的内容。

  • ls

    • ls命令就是list的缩写，缺省参数时ls用来打印出当前目录的清单，如果ls指定其他目录，那么就会显示指定目录里的文件及文件夹清单。 通过ls命令不仅可以查看linux文件夹包含的文件，还可以查看文件权限(包括目录、文件夹、文件权限)，查看目录信息等等。 ls命令的常用参数有-a, -l等。
